Department Of The Interior

Government website navigation redesign

As researchers we are interested in how users of the Dept. of Interior access the website to find information about how to volunteer in their community, information about climate change and how to enjoy their public lands. We want to know if the user finds the website useful, visually composed and easy to navigate.

We developed a user test and asked five participants to demonstrate their interaction with the site to find information and perform three basic tasks. This allowed our team to observe the pain points and the successes of navigating the site currently. This will give us the feedback we need to begin our redesign process to improve the user experience and the user interface.

Site Analysis

Existing User Flow

Existing user flow was analysed to determine what the actions were to discover different task paths for the site. We determined that the navigation was ambiguous and there was no clear path to volunteer activities or to search media offerings.

UI Heuristics

Existing Information Heirarchy

We analysed the site navigation components to determine what elements were working and how we could improve on the site. We determined that the information lacked a clear hierarchy. This leads to confusion and excessive clicking and back clicking to determine where the user is on the page and how the infomation can be accessed. There was also alot of navigation that opened new tabs or directed the user off-site which users found difficult to keep track of.

The current Department of the Interior website attempts to provide a wealth of information and interaction with no obvious hierarchy or clear navigation and is therefore confusing to the user. My goal is to make the navigation of the departments' multi-media and volunteer opportunities more intuitive.

A new user interface was developed with a clear user path and an intuitive navigation with clear informational hierarchy through the use of content chunking, clearly defined actions and an open and inviting interface to engage and inform the user

Proto Persona

We developed a proto-persona as part of our research to identify who will visit the site and why. We identified that the user would be interested in the Dept. data on climate change, volunteering and general information about the different media offerings from the Interior Dpartment.

Wire Framing

After analysing the site and organizing the information, It is time to create some wireframes to guide the new navigation heirarchy.

RWD Wireframes

In the wireframes prototype I was able to begin to address the user pain points from the testing such as off site page links and ambiguous information hierarchy. I initially created mobile first design to ensure continuity across both platforms.

Media Pages

UI Design-Style Guide

The style tile helps develop a visual UI system to implement into the site redesign and allows multiple designers to interact with key element and clarify the look and feel of the design. It is important that these elements convey the sense of importance and official purpose of the Department.

Color Palette and Typography

We chose to integrate colors and typography which conveyed the natural elements and reflect the colors represented in the department logo. The typography type-faces convey the departments official boldness through type faces that are no-frills but easy to read across small and large formats.

Components

A design system includes components created to aid navigation and increase user interaction and input through dropdown menus and input forms.
